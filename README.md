# 📊 Upskill Sales Dashboard (Power BI)

## 🚀 Overview
Built an interactive Power BI dashboard to track sales performance, marketing effectiveness, and student data for Upskill Digital Academy.

---

## 🛠 Tools Used
- Power BI  
- Excel  
- Power Query  
- DAX  

---

## 📊 Dashboard Highlights
- Total Revenue: 3M  
- Total Profit: 2715K  
- Total Students: 90  
- Marketing Budget: 347K  

---

## 📈 Key Analysis
- Revenue trend across months  
- Gender-wise contribution  
- Best performing ad types  
- Top 5 cities by revenue  
- Best age group for conversions  
- Course-wise filtering  

---

## 💡 Key Insights
- Majority revenue comes from adult age group  
- Certain ad types perform better in conversions  
- Revenue shows strong growth towards year-end  
- Top cities like Dharwad & Shimoga contribute most
